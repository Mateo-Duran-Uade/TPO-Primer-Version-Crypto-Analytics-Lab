# Crypto-Analytics Lab

- Sistema de gestión y análisis de activos digitales desarrollado en Python.
- Dentro del trabajo se utilizan estructuras como ciclos, listas, condicionales y matrices para el manejo y reporte de los datos.

## Funcionalidades
- Alta de activos digitales
- Baja de activos (solo si las unidades en tesorería son 0)
- Modificación de atributos de un activo
- Informe general ordenado por puntaje y nombre

## Datos de cada activo
- Nombre, Ticker, Valor USD, Volumen 24hs, Metodología, Unidades, Puntaje

## Archivos incluidos
 - main.py : Aca se encuentra el programa principal donde se ejecutan todos las funciones y se pone en marcha el menu y a la vez la precarga de acticos
 - funciones.py : En este archivo se encuentran todo el codigo sobre las funciones de todo el programa.

## Cómo ejecutar
- Requiere Python 
- Ejecutar el archivo main.py
